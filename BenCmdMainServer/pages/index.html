Finally (December 25, 2011)

Our new BenCmd system is finally almost done and ready for deployment to servers everywhere. At this point in time, since we're still working out the kinks in the system, we will be accepting servers on a closed basis. All banlists will be closely monitored during this period and any servers caught banning people against BenCmd rules will be immediately disqualified. In order to be eligible to gain access during this stage, servers must adhere to the following guidelines:
  • You must be running an official copy of BenCmd. Absolutely no exceptions will be made to this rule!
  • You may not be running any other banning plugin or any plugin marked as conflicting with BenCmd.
  • Development updates must be enabled.
  • Automatic banning based on TNT proximity must be disabled.
  • You must have at least 3 administrators/moderators.
  • All administators must have a clean record on the Minecrap server.
  • You must be a public server with either no whitelist or an online whitelist application.
    • Note: If you have a publicly available whitelist, ben_dude56 and Deaboy must be whitelisted for investigative reasons.
Please note that, should you choose to apply, a BenCmd developer will be visiting your server in order to verify compliance to the above guidelines. BenCmd developers reserve the sole right to disallow a server from using this system for any reason. To apply, please send an email to bencmdteam@gmail.com containing the following information:
  • Your server's CraftBukkit and BenCmd build IDs.
  • Your server's IP address or equivalent. (Please note that we do not accept Hamachi servers)
  • A list of all administrators or moderators on your server. These will be the only people allowed to globally ban, so don't forget yourself!
  • Your server's website URL, if applicable.
